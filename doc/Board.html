    class Board

    Board¶ ↑

    This class represents the TicTacToe board. It's responsibilities are to keep track of the board's status, update it when moves are made and determine if a player has won.

def initialize(dimension = 3)
        @dimension = dimension
        @board = []
        (dimension**2).times do
                @board.push(nil)
        end
end

    Status¶ ↑

    This method prints the status of the board to the command line.

    # File Board.rb, line 21
def status
        dim_loop() { |p|
                print "\n"
          dim_loop { |q|
                        index = p * @dimension + q
                        print @board[index] == nil ? "#{ index + 1 } " : "#{ @board[index].icon } "
                }
        }
        2.times { print "\n" }
end

def winner

        return scan_hor(p1) if scan_hor(p1)

        return scan_diag(p1) if scan_diag(p1)
